3. ✅ Jogo iniciará automaticamente

#### **Opção 3: CÓDIGO FONTE**
1. Instale Python 3.7+ 
2. Execute: `pip install pgzero`
3. Baixe o código fonte
4. Execute: `python Main.py`
5. ✅ Jogo iniciará automaticamente
